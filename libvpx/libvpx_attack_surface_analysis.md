# libvpx 攻击面分析

## 项目概述
libvpx 是 Google WebM 项目的核心库，提供 VP8 和 VP9 视频编解码功能。

## 主要攻击面

### 1. 解码器攻击面 (Decoder Attack Surface)
**风险等级**: 🔴 高危

#### 1.1 VP8/VP9 比特流解析
- **位置**: `vp8/`, `vp9/`
- **风险**: 
  - 恶意构造的比特流可能导致缓冲区溢出
  - 整数溢出/下溢
  - 无效的帧引用
  - 超大分辨率导致内存耗尽
- **现有覆盖**: `vpx_dec_fuzzer.cc`

#### 1.2 IVF 容器格式解析
- **位置**: `ivfdec.c`, `ivfdec.h`
- **风险**:
  - 文件头解析错误
  - 帧大小字段伪造
  - 时间戳溢出
- **现有覆盖**: 部分覆盖在 `vpx_dec_fuzzer.cc`

#### 1.3 图像格式处理
- **位置**: `vpx/vpx_image.h`, `vpx_scale/`
- **风险**:
  - 像素格式转换错误
  - YUV 平面计算错误
  - 色彩空间转换漏洞
- **现有覆盖**: ❌ 无专门fuzzer

### 2. 编码器攻击面 (Encoder Attack Surface)
**风险等级**: 🟡 中危

#### 2.1 原始图像输入
- **位置**: `vpx/vpx_encoder.h`
- **风险**:
  - 无效的图像尺寸
  - 不匹配的像素格式
  - 畸形的 YUV 数据
- **现有覆盖**: `vpx_enc_fuzzer.c`

#### 2.2 编码器配置
- **位置**: `vpx_codec_enc_cfg_t`
- **风险**:
  - 极端的比特率设置
  - 无效的时间层配置
  - 空间可伸缩性参数错误
- **现有覆盖**: 部分覆盖

### 3. 多线程处理
**风险等级**: 🟡 中危

- **位置**: `vp9/encoder/vp9_ethread.c`
- **风险**:
  - 竞态条件
  - 死锁
  - 数据竞争
- **现有覆盖**: ❌ 无专门fuzzer

### 4. 后处理模块
**风险等级**: 🟢 低危

- **位置**: VP8 后处理
- **风险**:
  - 去块效应滤波器错误
  - 去宏块效应错误
- **现有覆盖**: 部分覆盖在 `vpx_dec_fuzzer.cc`

### 5. 外部接口
**风险等级**: 🟡 中危

#### 5.1 控制接口
- **位置**: `vpx_codec_control_()`
- **风险**:
  - 无效的控制 ID
  - 类型不匹配的参数
  - 状态机错误
- **现有覆盖**: ❌ 无专门fuzzer

#### 5.2 帧缓冲管理
- **位置**: `vpx/vpx_frame_buffer.h`
- **风险**:
  - 外部帧缓冲区管理错误
  - 引用计数错误
- **现有覆盖**: ❌ 无专门fuzzer

### 6. 容器格式支持
**风险等级**: 🟡 中危

#### 6.1 WebM 容器
- **位置**: `webmdec.cc`, `webmenc.cc`
- **风险**:
  - Matroska/EBML 解析错误
  - 元数据注入
- **现有覆盖**: ❌ 无专门fuzzer

#### 6.2 Y4M 格式
- **位置**: `y4minput.c`, `y4menc.c`
- **风险**:
  - 头部解析错误
  - 帧边界错误
- **现有覆盖**: ❌ 无专门fuzzer

## 未覆盖的关键攻击面

### 优先级 1 (高)
1. ✅ **图像格式转换 Fuzzer** - 测试像素格式转换和色彩空间处理
2. ✅ **编码器控制接口 Fuzzer** - 测试 vpx_codec_control 的各种参数组合
3. ✅ **多帧序列 Fuzzer** - 测试帧间依赖和引用帧管理

### 优先级 2 (中)
4. **WebM 容器 Fuzzer** - 测试 WebM/Matroska 解析
5. **Y4M 格式 Fuzzer** - 测试 Y4M 输入处理
6. **外部帧缓冲 Fuzzer** - 测试自定义内存管理

### 优先级 3 (低)
7. **多线程压力 Fuzzer** - 测试并发编解码
8. **速率控制 Fuzzer** - 测试各种速率控制模式
9. **时间/空间可伸缩性 Fuzzer** - 测试 SVC 编码
